Previsualización

En este ejemplo se muestra el funcionamiento de la previsualización de imágenes mediante inputs de tipo file.

Formulario con previsualización de imagen

FastLight incorpora un pequeño script que nos permite realizar previsualizaciones de los ficheros de imagen que seleccionamos en los inputs de tipo file. Para usarlo, tan solo hay que añadir el fichero con <script src="js/Preview.js"></script> y añadir los ID:

  • file-with-preview al input de formulario.
  • preview-image a la imagen.

Para configurar las extensiones permitidas, podemos especificar el atributo data-extensions en el input de tipo file, donde las indicaremos separadas por "pipes", por ejemplo "jpg|jpeg|png|gif|webp". Ojo porque es sensible a mayúsculas y minúsculas.
