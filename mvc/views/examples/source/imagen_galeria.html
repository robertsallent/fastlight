En este ejemplo se muestran ejemplos de galerías de imágenes (también sirve para vídeo).

Galería básica

Este es el aspecto de una galería básica hecha en FastLight. Simplemente necesitamos un contenedor de clase gallery, aunque en este ejemplo le hemos añadido algunas clases más:

  • gallery: le da el aspecto y aplica el flex.
  • w75 (opcional): para que ocupe el 75% del ancho disponible.
  • centered-block (opcional): centrado en el documento.
  • my2 (opcional): aumentar el espacio vertical con otros elementos.

Las figuras dentro de la galería tienen las clases card y small. Encontraréis más información sobre figuras e imágenes en el ejemplo de figuras e imágenes:

  • card: forma de tarjeta
  • small: tamaño pequeño.

Ajustando la proporción con fit

Para ajustar imágenes de distintos tamaños en una galería flex sin que se deformen, añadiremos la clase fit a las imágenes. Esto las escalará para rellenar todo el espacio disponible (pero no veremos la imagen entera).

Es útil cuando tenemos imágenes de diferentes tamaños o distintas relaciones de aspecto en la misma galería. Para indicar la relación de aspecto, podemos usar las clases mencionadas en el ejemplo de imágenes y figuras.

Ajustando la proporción con scale

Otra opción es usar la clase scale para las imágenes, que añadirá bandas a los lados (o arriba y abajo) para coseguir que la imagen esté entera en el espacio disponible.
